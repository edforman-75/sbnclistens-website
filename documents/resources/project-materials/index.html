SBNC Listens

User Research Project - November 2025

Comprehensive Website Analysis Through 22 User Interviews

About This Project

The SBNC Listens project was initiated to understand member experiences with the Santa Barbara Newcomers Club website. Through 22 in-depth user interviews, we gathered qualitative and quantitative insights to identify critical areas for improvement.

Goal: Identify 7-8 priority pages to improve within the next 2 months, focusing on website-specific issues (excluding mobile app, Jotform workflows, and email systems).

Methodology: AI-assisted transcription using OpenAI Whisper, followed by multi-lens qualitative analysis through 9 design thinking frameworks.
