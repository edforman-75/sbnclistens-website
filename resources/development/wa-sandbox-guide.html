Wild Apricot Playground for SBNC

Wild Apricot (WA) does not offer dev/staging environments, test databases, or a one-click clone of your live site.

The only way to build or test a new site safely â€” without affecting SBNC's live production account â€” is to create a completely separate Wild Apricot account and use that as our playground.

Playground vs. Sandbox

Playground: A clean Wild Apricot account for building and testing without production data. This is what we'll create using a free trial or small paid account.

Sandbox: A copy of production data used for testing. WA does not provide this capability, though we can manually populate our playground with sample data from our nightly exports.

For SBNC, we're creating a playground since it won't have a copy of our prod data.

1. Why We Need a Separate Account

A separate WA account allows us to:

  • Build the new site layout without touching production
  • Test new workflows, custom fields, membership levels, and event policies
  • Experiment with theme overrides, CSS, JavaScript, widgets
  • Test API integrations and the chatbot automation
  • Prototype new admin pages before rollout

Nothing we do in the playground affects real SBNC members, events, or transactions.

2. Ways to Create a Playground

There are three viable approaches for SBNC:

A. Free 30-Day Trial (Most Cost-Effective)

Create a new WA trial account under any email address.

Advantages

  • Full functionality (pages, themes, widgets, API)
  • API access is fully enabled during trial
  • Unlimited site editing and configuration
  • Perfect for building the new site

Limitations

  • Only 30 days
  • Cannot extend unless WA support intervenes
  • MAX 250 contacts on the lower-tier trial
  • Cannot import SBNC's full 900-person membership
  • Does not support the Custom HTML Gadget â€” quite limiting for advanced customizations

Workaround for data
We already generate nightly JSON data exports for the chatbot prototype. We can easily reuse those to repopulate the playground nightly with anonymized or reduced data (e.g., a 250-member sample).

B. Use Referral Discount to Reduce Cost

WA offers referral credits (typically $100 credit for both organizations). If SBNC opens a new account through a referral, the playground account cost drops.

C. Create a Paid "Small" Account for 60 Days

If 30 days is too short, the next option is to temporarily pay for a small WA subscription.

3. Can We Copy Our Existing Site?

WA does not provide any cloning tools, but we can reconstruct a staging site manually using:

  1. Our nightly JSON data export pipeline
  2. Manual page rebuild (clean rebuild is often desirable)
