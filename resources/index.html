SBNC Listens Resources

What's in This Section

The Resources section contains everything you need to understand the SBNC Listens project: user research findings, website analytics, technical documentation about Wild Apricot and platform options, design inspiration from other organizations, and information about the project team.

Who Should Use These Resources?

  • Technology Committee Members: Technical docs and platform analysis
  • Board Members: Research findings and action plans
  • Design Task Force: WA site examples and design inspiration
  • All Team Members: Project overview and team roster
