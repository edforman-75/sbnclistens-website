How to Record Your SBNC Interview

Quick Recommendations

In-Person Interview?
‚Üí Use your phone's voice memo app (iPhone or Android)

Remote Interview?
‚Üí Use Zoom recording

Both are free and easy!

Platform-Specific Instructions

üì± iPhone Voice Memos

  1. Open Voice Memos app (red icon with waveform)
  2. Tap red record button
  3. Place phone on table between speakers
  4. Tap red square to stop
  5. Share ‚Üí Save to Files or email

Tips:

  • Place 6-12 inches from speakers
  • Recording continues even if screen locks
  • Files save as .m4a (perfect!)

Detailed Guide: https://support.apple.com/en-us/HT206775

üì± Android Voice Recorder

Samsung/Google Pixel:

  1. Open "Voice Recorder" or "Recorder" app
  2. Tap red microphone button
  3. Place phone on table
  4. Tap stop when finished
  5. Share ‚Üí Upload to Drive or email

Other Android:
Download "Easy Voice Recorder" (free from Play Store)

Tips:

  • Check you're not in airplane mode
  • Disable battery saver during recording

üíª Zoom Recording (Remote Interviews)

  1. Start Zoom meeting (free account works)
  2. Send join link to interviewee
  3. Click "Record" ‚Üí "Record to this Computer"
  4. Interview auto-saves when you end meeting

Where it saves:

  • Mac: /Users/[YourName]/Documents/Zoom
  • PC: C:\Users\[YourName]\Documents\Zoom

Detailed Guide: https://support.zoom.us/hc/en-us/articles/201362473

üíª Mac QuickTime

  1. Open QuickTime Player
  2. File ‚Üí New Audio Recording
  3. Click red record button
  4. Click stop when done
  5. File ‚Üí Save

Detailed Guide: Apple QuickTime Tutorial

üíª Windows Voice Recorder

  1. Click Start ‚Üí Type "Voice Recorder"
  2. Open Voice Recorder app
  3. Click microphone button to start
  4. Click stop when finished
  5. Right-click recording ‚Üí "Open file location"

Where it saves: C:\Users\[YourName]\Documents\Sound recordings

Detailed Guide: Microsoft Guide

Getting Consent

Use this script:

"I'd like to record our conversation so we can transcribe and review it accurately. The recording will only be used by our team to analyze themes. Your name won't be attached to quotes. And if you'd prefer complete anonymity, just let me know. Is it okay if I record?"

Wait for clear verbal "Yes" before starting.

Quick Tips

  • ‚úÖ Test first (30-second test recording)
  • ‚úÖ Place device 6-12 inches from speakers
  • ‚úÖ Phone on silent (recording still works!)
  • ‚úÖ Check storage (need 15-30 MB)
  • ‚úÖ Plug in if possible for longer recordings

After Recording

  1. STOP the recording
  2. Listen to first 10 seconds to verify
  3. Name the file: SBNC_Interview_[Name]_Nov[Date]
  4. Submit via online form
